# BARSKY PROTOCOL — 7000 Words Mission

A Leitner-system-based vocabulary learning web application featuring 7,000 essential English words for the Taiwan CEEC exams.

## Features
- **Leitner System Implementation**: 5 boxes with automated scheduling based on review performance.
- **Full 7000 Dataset**: Comprehensive vocabulary spanning 6 complexity levels.
- **Rich Card Data**: Includes KK Phonetics, Chinese translations, and contextual example sentences.
- **Commonality Indicators**: Star ratings (⭐⭐⭐) based on real-world usage frequency.
- **Interactive UI**: Mobile-friendly glassmorphism design with auto-pronunciation.

## Protocol Instructions
1. **New Vocabulary**: Study the daily batch of 20 new words.
2. **Review Operations**: Process cards due in each box.
3. **Box Progression**:
   - Correct answers move cards to higher boxes (longer review intervals).
   - Incorrect answers return cards to Box 3 for reinforcement.

## Deployment
Built as a static progressive web app (PWA), designed for GitHub Pages.
